# Fundamentos do Flask - Meu Guia de Estudos

Este repositório documenta minha jornada de aprendizado dos fundamentos do Flask, seguindo principalmente o guia de início rápido oficial. O foco é dominar a criação de aplicações web com Python, desde a configuração básica até a estruturação de projetos mais organizados.

## 🎯 Objetivo

Consolidar o conhecimento essencial para construir aplicações web com Flask, entendendo na prática os conceitos de rotas, templates, requisições e respostas.

## 📚 Principais Tópicos Estudados

Consultar a documentação oficial https://flask.palletsprojects.com/en/stable/quickstart
